# 🎨 Digital Art Gallery

A simple and elegant web app to upload and browse digital artworks. Built with HTML, Tailwind CSS, and vanilla JavaScript. Artworks are saved in your browser using localStorage and you can also fetch beautiful artworks from the Metropolitan Museum of Art API.

## ✨ Features

- Upload your own artworks (title, artist & image URL)
- Fetch popular artworks from the Met Museum API
- Responsive gallery grid layout
- Persistent storage using localStorage
- Clean and modern UI with Tailwind CSS
- Fully functional without any backend

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript
- Metropolitan Museum of Art Public API
- localStorage

## 🚀 How to Run

1. Download the two files:
   - `index.html`
   - `script.js`

2. Place both files in the **same folder**.

3. Open the `index.html` file in your web browser.

> No installation or server needed — it works directly in the browser!
